# FarmerBlocks — Static Website

This folder contains a minimal static website for FarmerBlocks (modular hydroponic containers).

Try it locally:

```bash
# from this folder
python3 -m http.server 8000
# then open http://localhost:8000
```

Files:
- index.html — main page
- styles.css — site styles
- script.js — small interactivity helpers
- assets/logo.svg — placeholder logo

Next steps you might want:

- Hook contact form to a backend or use Formspree/Netlify forms.
